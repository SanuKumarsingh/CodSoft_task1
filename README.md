# ✅ To-Do List App — CodSoft Task 1

A simple command-line **To-Do List Application** built in Python as part of the **CodSoft Python Programming Internship (Task 1)**. Lets you add, view, and complete tasks — with data saved persistently using a JSON file.

## 🔗 Repository

[github.com/SanuKumarsingh/CodSoft_task1](https://github.com/SanuKumarsingh/CodSoft_task1)

## ✨ Features

- View all tasks with their completion status
- Add new tasks to the list
- Mark tasks as complete
- Data is saved to `todo_list.json` — persists between sessions
- Handles missing file gracefully on first run

## 📁 Project Structure

```
CodSoft_task1/
├── ToDoList.py       # Main Python script
└── todo_list.json    # Auto-generated task storage file
```

## 🚀 Getting Started

**Requirements:** Python 3.x

```bash
git clone https://github.com/SanuKumarsingh/CodSoft_task1.git
cd CodSoft_task1
python ToDoList.py
```

## 🛠️ Menu Options

| Option | Action                        |
|--------|-------------------------------|
| `1`    | View all tasks with status    |
| `2`    | Add a new task                |
| `3`    | Mark a task as complete       |
| `4`    | Save and exit                 |
